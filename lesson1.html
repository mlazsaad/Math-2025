الأعداد الطبيعية (Natural Numbers)

هي الأعداد الموجبة المستخدمة في العد: {1, 2, 3, ...}

خصائص الأعداد الطبيعية:

  • موجبة دائماً
  • لا تتضمن الكسور
  • لا تتضمن الصفر (في بعض التعريفات)

الأعداد النسبية (Rational Numbers)

Q = { p/q | p, q ∈ ℤ, q ≠ 0 }

الأعداد التي يمكن التعبير عنها كنسبة بين عددين صحيحين، تشمل:

  • الكسور المركبة: 3/4 ،-2/5
  • الأعداد العشرية الدورية: 0.333...

الأعداد غير النسبية (Irrational Numbers)

𝕀 = { x ∈ ℝ | x ∉ ℚ }

أعداد لا يمكن التعبير عنها كنسب صحيحة، مثل:

  • الجذر التربيعي لـ2 (√2)
  • العدد النيبيري (e)

الأعداد التخيلية (Imaginary Numbers)

𝕀 = { bi | b ∈ ℝ, i = √-1 }
3i

أُسست لحل المعادلات المستعصية في ℝ مثل:

x² + 1 = 0 ⇒ x = i, -i
